          第1集
          华盛顿全城的警力都在全力追捕涉嫌毒杀参议员霍金斯并枪杀司法部长康诺利的前FBI特别探员伊利莎白•基恩及其同伙雷蒙德•雷丁顿。城里所有的地面、空中交通均被封锁，FBI探员莱斯勒负责此次追捕行动，他要让嫌犯插翅难逃。全城戒严早在雷丁顿的预料之中，最危害的地方才最安全，他并不打算冒险出城，而是要在城里寻求帮助。雷丁顿早有安排，在同伙的接应下利用隧道盲区顺利摆脱警方追踪。随后两人换上准备好的警服，开着假警车混入大街的车流中。原FBI特别行动小组组长、FBI副局长哈罗德•库珀因涉嫌帮助伊利莎白而接受调查。即使他说出秘社通过康诺利控制他，也只是更增加自己的嫌疑，拿不出任何证据证明自己的说法。他现在唯一的要求就是希望莱斯勒能释放被无辜牵涉进来的夏琳，他深爱的妻子。目前暂时负责行动小组的莱斯勒也不知道该相信什么人。行动小组里纳瓦比和阿兰姆都不相信伊利莎白会是俄罗斯间谍，莱斯勒虽然也不愿相信，但他的职责让他必须追捕伊利莎白和雷丁顿。整个小组就在这种矛盾的气氛中运作着，完全没有了以前的合作氛围。
          第2集
          华盛顿特区的俄罗斯大使馆如临大敌。外面警车及武装车辆将使馆包围，双方隔着铁门对峙。莱斯勒虽然恼怒自己放走了一个俄罗斯间谍，但他也知道贸然冲进使馆无异于武装入侵俄罗斯，很可能引发一场世界大战。使馆审讯室内，伊利莎白对前来盘问的小角色毫无交谈的兴趣，直接要求与情报负责人对话。不一会，俄联邦安全局驻华盛顿高级情报官亚历山大•罗斯科维奇进来，随手将审讯室内的监控、监听设备一一关闭。与此同时，俄罗斯大使康尼正在与自己的情人幽会，急促的电话声打断了他的好兴致。他从浴室里出来，看到是使馆的电话，愤愤的挂断。背后突然有人打招呼，转头就看到雷丁顿拿着手枪坐在椅子里。康尼刚想叫人，可雷丁顿马上就让他知道俄国大使与美国参议员老婆偷情的桥段会被媒体炒翻天。康尼不得不妥协，而雷丁顿的要求就是马上联系大使馆，他要实时知道俄大使馆里发生的所有事情，以及莫斯科高层的决定。亚历山大对每一名在美洲的现役和非现役情报员都了如指掌，但他却从不知道有个叫玛莎•罗斯托娃的俄罗斯特工。
          第3集
          汤姆和莱斯勒的见面并不愉快。莱斯勒对汤姆余恨未消，双方也没有任何信任可言。汤姆不相信伊利莎白是俄罗斯间谍，想在一些FBI无法出面的场合提供帮助。但莱斯勒并不领情，他绝不会违背自己的原则与这个曾经的通缉犯合作。弗地安特企业是全球最大的综合性农作物销售商，产品销到全球各地。因其转基因的玉米、大豆种子有良好的抗虫基因而广受欢迎，但也因其价格过高受到小农户的抵制。一群自称“收割者”的激进份子通过内应袭击了爱荷华州的弗地安特企业分厂，不仅造成人员伤亡，还盗取了最重要的转基因专利资料数据。如果这份资料被公布，弗地安特企业的产品就会被大量仿制，导致其股票大跌。而弗地安特企业还有另一重隐秘身份，秘社的洗钱机构。深夜，爱荷华州际公路上一辆货柜车在疾驰，车上的集装箱里正是雷丁顿和伊利莎白。雷丁顿没有按原定计划跨海去西班牙，而是打算反守为攻。他相信秘社的重要头目“主任”彼得•卡奇奥普罗斯因为在支点事情上判断失误，已经引起秘社其他成员不满。随着那些顶尖记者对支点数据的挖掘，秘社成员会慢慢曝光。
